
Receitinhas Viciantes - Hugo site
-------------------------------
This package contains a Hugo site ready for deploy to Netlify.
Steps to publish:
1. Create a GitHub repository and push the contents of this folder.
2. On Netlify, choose 'Import from GitHub', select the repo.
3. In Netlify build settings: Build command: `hugo --gc --minify` ; Publish directory: `public` ; Branch: main
4. Deploy.
